04 · Foundations

Typography

Inter is the main typeface — SIL Open Font License, on Google Fonts, built for on-screen UI and dense data. Lexend (also OFL) is wired in as an optional easy-reading mode for body text: add class="reading" to any container. Inter 400/500 for body and UI, 600/700 for headings, 800 for the wordmark; both cover the Latin script used in Indonesian.

06 · Handoff

Copy-paste tokens

Drop into :root. The green scale and semantic colours are derived; the two identity colours trace to the emblem.

/* identity — anchored to the Pancacita seal */
--primary:        #288050;  /* emblem-hue green, desaturated */
--primary-hover:  #1f6b41;
--gold:           #fcd800;  /* accent — dark text only, small areas */
--gold-muted:     #e2ba36;

/* green scale — derived (hue 147°) */
--green-50:#f1f9f5;  --green-100:#dff1e8; --green-200:#c0e3d0; --green-300:#8dceaa;
--green-400:#4cbd7f; --green-500:#2e9e60; --green-600:#288050; --green-700:#1f6b41;
--green-800:#195737; --green-900:#163b29;

/* surfaces & text — derived */
--bg:#f1f9f5; --surface:#ffffff; --surface-tint:#e6f4ed; --border:#cfe2d9;
--ink:#163b29; --ink-soft:#5a7068; --on-primary:#ffffff;

/* type (both SIL OFL) + radius */
--font-main:"Inter",system-ui,sans-serif;
--font-reading:"Lexend","Inter",system-ui,sans-serif;
--r-sm:3px; --r-md:5px; --r-lg:8px; --r-xl:10px; --r-pill:6px;
Three honest caveats. (1) The green is the seal's hue, deliberately desaturated for long-session comfort — it is not the emblem's exact 100%-saturation green; the gold is sampled from your uploaded image. If you need official values, verify against Pemerintah Aceh's gazetted brand/qanun spec, since the uploaded file may be a redrawn version. (2) The 50–900 scale and all surface/text colours are computed, not from the seal — a sound starting point, not gospel. (3) Gold is accent-only: never white text on it, never gold text on white.
