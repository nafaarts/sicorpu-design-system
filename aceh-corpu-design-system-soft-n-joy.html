03 · Foundations

Contrast & accessibility

Ratios below are computed with the exact WCAG 2.x formula. The headline finding: white text on the primary teal fails AA — use teal-700 or darker for white text.

CombinationPreviewRatioVerdict
White on primary #53a687 Aa 2.93Fail
White on teal-700 Aa 4.76AA
Ink #14463a on mint bg Aa 9.87AAA
Ink on light lime Aa 8.76AAA
Ink on yellow-green Aa 6.82AA
Ink on peach accent Aa 6.09AA
Primary teal on mint bg Aa 2.71Fail*

*Teal text on mint is fine for large/bold display only (≥24px or ≥18.66px bold → 3.0 threshold). For body text on mint, use the ink color.

04 · Foundations

Typography

Inter is the system's main typeface — SIL Open Font License, on Google Fonts, built for on-screen UI and dense data. Lexend (also OFL) is wired in as an optional easy-reading mode for body text: add class="reading" to any container. Use Inter weights 400/500 for body and UI, 600/700 for headings, 800 for the wordmark. Both cover the Latin script used in Indonesian; Inter also carries broad extended-Latin and tabular figures for forms and tables.

06 · Handoff

Copy-paste tokens

Drop these CSS custom properties into :root. Source colors are the labelled values; scale and semantic colors are derived.

/* core — from source image labels */
--color-primary:    #53a687;
--color-secondary:  #c1d96c;
--color-tertiary:   #def294;
--color-accent:     #f2b885;
--color-background: #e8faf4;

/* primary scale — derived */
--teal-50:#f1f8f5; --teal-100:#e0efe9; --teal-200:#beddd1;
--teal-300:#98cab7; --teal-400:#72b69d; --teal-500:#53a687;
--teal-600:#479377; --teal-700:#3a7f68; --teal-800:#2c6a56; --teal-900:#1d5243;

/* semantic — derived */
--text-primary:   #14463a;
--text-secondary: #3a6b5c;
--surface:        #ffffff;
--border:         #cde8de;
--on-primary:     #ffffff;  /* use on --teal-700+ only */

/* type — both SIL Open Font License, on Google Fonts */
--font-main:    "Inter", system-ui, sans-serif;
--font-reading: "Lexend", "Inter", system-ui, sans-serif; /* opt-in */

/* radius — tightened for institutional tone */
--r-sm:3px; --r-md:5px; --r-lg:8px; --r-xl:10px; --r-pill:6px;
Three honest caveats. (1) This green palette is carried over from the source image as a placeholder identity — it is not Aceh Corporate University's verified official brand. If an official logo/color/motto exists, replace these tokens with it. (2) The teal and peach swatches render more saturated in the source PNG (#00a887, #ffba83) than their printed labels — built on the labels; swap if you prefer the punchier render. (3) The 50–900 scale and all semantic colors are interpolated, not extracted — a sensible starting point, not gospel.
